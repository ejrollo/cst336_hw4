What is Data Science?

Data Science encompasses many disciplines including computer science, statistics and business/domain knowledge.

This site will give you a general concept of Data Science, so that you can explore further throughout the web.

Data Science is a blend of various tools, machine learning principles, and algorithms.

Put together, these aspects will allow you to better understand the datasets of a domain by discovering hidden partterns within the raw data.

What differentiates a Data Scientist from a traditional Statistician is with the preditictive tools used by Data Scientists. Traditional Statisticians will explain a data set, while a Data Scientist will not only explain, but use tools such as machine learning to determine the occurance of a future event.
